# Welcome!

This is your GitHub Pages site powered by Jekyll. You can edit this page by modifying the `index.md` file.

## Getting Started

- Check out the [Jekyll docs](https://jekyllrb.com/) to learn more
- Customize the theme and layout in `_config.yml`
- Add more pages by creating new `.md` files in the root directory
- Create blog posts by adding files to the `_posts` folder

## Next Steps

1. Edit the `_config.yml` file to update your site title and description
2. Replace this content with your own
3. Add your logo and customize colors in the theme settings
4. Push your changes to GitHub and your site will be live!
